## jsDoc

- `@property` 声明一个带有深层结构的对象的属性

	```js
	// 假设传入的参数格式为：p: {a: 1, b: '1234'}
	/**
	 *  @param {Object} p
	 *  @property {number} p.a
	 *  @property {string} p.b
	 */
	 function test(p) { ... }
	```
- `@exports` 该对象由js module export
- `@memberof` 标记父类

	- Syntax
	
		- `@memberof <parentNamepath>`
		- `@memberof! <parentNamepath>`

	- Example

		```js
		/** @namespace */
		var Tools = {};
		
		/** @memberof Tools */
		var hammer = function() {
		};
		
		Tools.hammer = hammer;
		```
